# PandoraPro Admin (frontend-only)

Bản demo frontend theo layout Figma/screenshot, **không phụ thuộc backend**.

## Chạy local

```bash
npm install
npm run dev
```

- Mặc định chạy tại `http://localhost:5173`.
- Login là mock (chấp nhận mọi email/password), dữ liệu được seed + lưu trong `localStorage`.

## Ghi chú

- Dữ liệu mock: `src/admin/mocks/db.ts`
- Layout: `src/admin/layout/AdminLayout.tsx`
- Pages: `src/admin/pages/**`
